# split-file-by-sutra步驟說明:

1.將原資料放進/split-file-by-sutra/data的路徑目錄中。

2.開啟Command Prompt於/split-file-by-sutra路徑中執行node splitSutra.js。

3.返回"split-file-by-sutra"資料夾即可看到"splited"資料夾，打開後就可看見所有按sutra id命名的目錄及其檔案。
